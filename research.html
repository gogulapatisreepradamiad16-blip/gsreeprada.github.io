Context & Theoretical Grounding

Traditional structural facades operate statically, often failing to adapt to fluctuating microclimatic factors. This registered system abstracts the "Flower of Life" geometry into scalable modular building lattices. Each polygonal block features mathematically derived curved voids designed to adapt safely to external environments without separate framework components.

The Tectonic Workflow & Prototyping Cycles

The manufacturing process bridges digital scripting with advanced CNC milling and casting workflows. To make the assemblies strong yet lightweight, a 3-layer sandwich joint system was created to build continuous, frameless structural lattices across variable geometric thicknesses (ranging from small 2mm acrylic components up to large 20mm timber architectural panels).

Flagship Project 02 // Empirical Modeling

Quantifying Jaali Geometry Relationships with Daylight Factor (DF)

70% Mathematical Modeling20% Regression Graphs10% Envelope Metrics

This project approaches building physics through strict **Environmental Design Research** rather than aesthetic styling. It explicitly charts modern glass curtain wall deficiencies (which trigger heavy reliance on mechanical cooling and interior blinds) against mathematically managed solar control facades.

Fig 2.1: Simulation performance graph demonstrating calculation of solid-to-void percentages against climate boundaries.

The Empirical Workflow

By extracting specific regression equations, architects can calculate the exact solid-to-void percentages needed to achieve a targeted target Daylight Factor (DF) inside interior rooms. This system balances absolute glare protection with natural airflow parameters, transforming traditional craft principles into highly predictable, energy-saving facade applications.

Secondary IP Track // Patent Specification

Dynamic Gear Actuated Flexure Casting Infrastructure

A specialized innovation focused on making continuous, dynamic adjustments to casting material void ratios. This system deploys a modular outer shell made of 6061-T6 aluminum alloy that houses a set of identical flexure walls arranged in an equilateral triangular configuration.

Controlled via a master bevel gear assembly linked to linear lead screw actuators, the 2mm AISI 1095 spring-steel flexure walls deform precisely inward or outward. Coated with Polytetrafluoroethylene (PTFE) for complete chemical compatibility, this infrastructure allows engineers to cast custom architectural screen panels with variable solid-to-void gradients using a single, adjustable mold system.
